# 价值提取效率评估报告

## 评估方法

**价值提取效率** = Σ(选中新闻的价值) / Σ(理想Top20的价值) × 100%

- 100% = 完美，选中的就是价值最高的20条
- 效率越高 = 方案的信息提取能力越强

## 评估步骤

1. 对每个用户×日期的400+条新闻全量打分
2. 建立价值排行榜，得到理想Top20及其总价值
3. 计算各方案选中新闻的总价值
4. 效率 = 选中价值 / 理想价值 × 100%
